## Short Description (under 150 chars)
Monitor macOS RAM, swap, and memory pressure in real-time with color-coded progress rings on your Stream Deck.

## Full Description
Keep an eye on your Mac's memory health without opening Activity Monitor. Mac Memory Monitor displays real-time RAM and swap usage directly on your Stream Deck with intuitive color-coded progress rings that change from green to yellow to red as usage increases.

**8 Memory Metrics Available:**
- Swap Usage – Know when your Mac is swapping to disk
- RAM Usage – Total memory consumption at a glance
- App Memory – Memory used by running applications
- Wired Memory – Locked memory that can't be paged
- Compressed Memory – macOS memory compression indicator
- Cached Files – File cache usage
- Physical RAM – Total installed memory display
- Custom Metric – Choose any metric per button

**Why You Need This:**
- Instantly spot memory pressure before slowdowns occur
- Color-coded alerts: green (healthy), yellow (moderate), red (critical)
- Clean progress ring visualization updates in real-time
- Perfect for developers, content creators, and power users monitoring system resources

**Requirements:**
- macOS 10.15 (Catalina) or later
- Stream Deck 6.6 or later
